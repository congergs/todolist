Hello from todolist!

If this app instance is in developer mode, it loads code and assets from the 

apps/devel/todolist.mburak.user.app.sobamail.com

folder in your profile folder. Let's call this the develroot. The full path for the develroot is: 

C:/Users/burak/AppData/Local/Soba/Mail/m-AGSPNQ\apps\devel\todolist.mburak.user.app.sobamail.com

The mutator (backend) entry point can be found at: {develroot}/Mutator.mjs

The GUI entry point (frontend) can be found at: {develroot}/assets/index.html

To open your profile folder, go to Tools > Profile Folders screen and click "Open Folder".
